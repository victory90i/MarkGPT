## How to Contribute

- Improve explanations of neural network concepts
- Add more examples with visualizations (activation functions, forward/backward pass)
- Create debugging guides for common errors
- Build step-by-step implementations from scratch

## Key Guidelines

- ✅ Show mathematical derivations alongside code
- ✅ Include visualizations of network behavior
- ✅ Test all implementations thoroughly
- ✅ Add comments explaining each matrix operation
