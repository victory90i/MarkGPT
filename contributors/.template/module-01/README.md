# Module 01 Template Structure

This is a reference template showing the expected folder structure for **Module 01: Foundations of AI & Language**.

## Folder Layout

```
module-01/
├── lessons/
│   ├── L01.1_what_is_ai.ipynb
│   ├── L01.2_history_of_nlp.ipynb
│   ├── L01.3_language_as_data.ipynb
│   └── ...
├── exercises/
│   ├── E01.1_discussion_questions.ipynb
│   ├── E01.2_history_timeline.ipynb
│   └── ...
└── projects/
    ├── P01.1_interview_project.ipynb
    └── ...
```

## How to Use This Template

When you work in `contributors/YOUR-USERNAME/module-01/`, **mirror this structure exactly**:

1. Create `lessons/` folder for lesson improvements
2. Create `exercises/` folder for additional practice problems
3. Create `projects/` folder for mini-projects or demos

## Types of Files You Might Add

### 📚 Lessons
- `improved_chapter_name.ipynb` — Enhanced version of an existing lesson
- `extra_examples.ipynb` — Additional code examples and visualizations
- `student_notes.md` — Study guides and summaries

### 🎯 Exercises
- `practice_problems.ipynb` — Additional problem sets
- `solutions_walkthrough.ipynb` — Step-by-step solutions with explanations
- `beginner_exercises.ipynb` — Simplified practice for struggling students

### 🚀 Projects
- `capstone_mini_project.ipynb` — End-to-end project combining concepts
- `real_world_example.ipynb` — Application of concepts to real data

## 📝 File Naming Convention

- Use **descriptive names** that explain the content
- Use **snake_case** for Python files: `data_analysis.py`
- Use **UPPERCASE** for markdown: `LEARNING_NOTES.md`
- Use **descriptive Jupyter names**: `improved_linear_regression.ipynb`

## ✅ Checklist Before Submitting

- [ ] All Jupyter notebooks execute without errors
- [ ] Code includes explanatory comments
- [ ] Markdown cells explain complex concepts
- [ ] Examples are clear and relevant
- [ ] No hardcoded file paths (use relative paths instead)
- [ ] Files are saved in the correct subfolder (lessons/exercises/projects)
