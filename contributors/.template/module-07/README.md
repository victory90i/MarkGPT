## How to Contribute

- Explain pretraining objectives (LM loss, MLM loss)
- Add training monitoring and visualization
- Create guides for scaling training
- Build training pipelines and utilities

## Key Guidelines

- ✅ Include loss landscape visualizations
- ✅ Explain FLOPs and training efficiency
- ✅ Add checkpointing and recovery strategies
- ✅ Provide debugging for training issues
