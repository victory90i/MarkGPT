# Module 02 Template Structure

This is a reference template showing the expected folder structure for **Module 02: Python & Math Essentials for ML**.

## Folder Layout

```
module-02/
├── lessons/
│   ├── L02.1_python_basics.ipynb
│   ├── L02.2_numpy_arrays.ipynb
│   ├── L02.3_pandas_dataframes.ipynb
│   └── ...
├── exercises/
│   ├── E02.1_python_challenges.ipynb
│   ├── E02.2_numpy_practice.ipynb
│   └── ...
└── projects/
    ├── P02.1_data_analysis_project.ipynb
    └── ...
```

## How to Use This Template

When you work in `contributors/YOUR-USERNAME/module-02/`, **mirror this structure exactly**:

1. Create `lessons/` folder for lesson improvements
2. Create `exercises/` folder for additional practice problems
3. Create `projects/` folder for mini-projects or demos

## Types of Contributions Needed

### 📚 Lessons
- Clarify NumPy concepts with additional visualizations
- Provide more pandas examples
- Add mathematical intuition explanations
- Include debugging tips for common errors

### 🎯 Exercises
- Create tiered difficulty problems (beginner → advanced)
- Add "fix the code" challenges
- Provide solution walkthroughs
- Include interactive exercises

### 🚀 Projects
- Real data analysis projects
- Data cleaning and transformation workflows
- Combine Python, NumPy, and Pandas in one project

## 📝 File Naming Convention

- Use **descriptive names**: `numpy_advanced_indexing.ipynb`
- Use **snake_case** for Python: `matrix_operations.py`
- Use **UPPERCASE** for guides: `NUMPY_TROUBLESHOOTING.md`

## ✅ Before Submitting

- [ ] All code is tested and runs successfully
- [ ] Explanations are clear for beginners
- [ ] Code includes comments for complex sections
- [ ] Mathematical notation is correct (if applicable)
- [ ] No hardcoded paths; use relative paths
