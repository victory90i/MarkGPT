# Module 05 Template Structure

This is a reference template showing the expected folder structure for **Module 05: NLP Foundations - Text as Data**.

## Folder Layout

```
module-05/
├── lessons/
│   ├── L05.1_tokenization.ipynb
│   ├── L05.2_word_embeddings.ipynb
│   ├── L05.3_text_preprocessing.ipynb
│   └── ...
├── exercises/
│   ├── E05.1_tokenization_challenges.ipynb
│   ├── E05.2_embedding_exploration.ipynb
│   └── ...
└── projects/
    ├── P05.1_text_classification.ipynb
    └── ...
```

## How to Contribute

- Explain tokenization strategies with examples
- Add visualizations of word embeddings
- Create text preprocessing pipelines
- Build practical NLP applications

## Key Guidelines

- ✅ Use real text examples (news, social media, books)
- ✅ Visualize embeddings in 2D/3D space
- ✅ Show before/after text preprocessing
- ✅ Include handling of non-English text
