## ❓ FAQ About This Template

**Q: Can I nominate someone else?**
A: Absolutely! Create a pull request with a completed spotlight template.

**Q: Can I nominate myself?**
A: Yes, but maintainers can also create spotlights based on outstanding contributions.

**Q: How often are spotlights published?**
A: Typically monthly, featuring contributors who have made significant impact.

**Q: Will I get credit/recognition?**
A: Yes! Spotlighted contributors are:
- Featured in this template's hall of fame
- Mentioned in project README
- Added to CONTRIBUTORS.md
- Recognized with a badge/tier upgrade on GitHub
