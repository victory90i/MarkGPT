# Citation Guide

## About MarkGPT

If you use MarkGPT in research or production, please cite this work:

## BibTeX Citation

```bibtex
@repository{markgpt2024,
  title={MarkGPT: A 60-Day LLM Curriculum with Multilingual Support},
  author={IWS Technical},
  year={2024},
  url={https://github.com/iwstechnical/MarkGPT-LLM-Curriculum},
  note={Educational curriculum for understanding and building language models}
}
```

## Academic Paper (if available)

```bibtex
@article{markgpt2024paper,
  title={MarkGPT: Teaching LLM Fundamentals Through a Complete, Multilingual Curriculum},
  author={Author Name},
  journal={arXiv preprint arXiv:XXXX.XXXXX},
  year={2024}
}
```

## Using Our Models

If you use MarkGPT pre-trained models, cite both the curriculum and the original papers:

```bibtex
% MarkGPT curriculum
@repository{markgpt2024,
  title={MarkGPT: A 60-Day LLM Curriculum with Multilingual Support},
  author={IWS Technical},
  year={2024},
  url={https://github.com/iwstechnical/MarkGPT-LLM-Curriculum}
}

% Transformer architecture
@inproceedings{vaswani2017attention,
  title={Attention is All You Need},
  author={Vaswani, Ashish and others},
  booktitle={NeurIPS},
  year={2017}
}

% For multilingual models, also cite:
@article{lewis2020multilingual,
  title={Multilingual Denoising Pre-training for Neural Machine Translation},
  author={Lewis, Mike and others},
  journal={arXiv preprint arXiv:2001.08210},
  year={2020}
}
```
