# Neural Networks from First Principles

## Why This Matters

Deep learning isn't magic—it's just repeated application of the chain rule. Understanding backpropagation from scratch builds the intuition you'll need to debug training failures and design novel architectures. Researchers who understand the math can innovate; those who don't will always be limited.

## The Core Insight: Chain Rule

Any neural network is a **composition of functions**:

$$y = f(g(h(x)))$$

To find how changing $x$ affects $y$, we use the chain rule:

$$\frac{dy}{dx} = \frac{dy}{df} \cdot \frac{df}{dg} \cdot \frac{dg}{dh} \cdot \frac{dh}{dx}$$

**Backpropagation is just the chain rule applied systematically.**

## A Simple Neural Network

```python
import numpy as np

class SimpleNeuralNet:
    """Single hidden layer network: x -> h -> y"""
    
    def __init__(self, input_size=2, hidden_size=3, output_size=1):
        # Initialize weights randomly (small values prevent saturation)
        self.W1 = np.random.randn(input_size, hidden_size) * 0.01
        self.b1 = np.zeros((1, hidden_size))
        self.W2 = np.random.randn(hidden_size, output_size) * 0.01
        self.b2 = np.zeros((1, output_size))
    
    def forward(self, X):
        """Forward pass: compute predictions"""
        self.z1 = X @ self.W1 + self.b1  # Linear: (batch, hidden)
        self.a1 = np.maximum(0, self.z1)  # ReLU activation
        self.z2 = self.a1 @ self.W2 + self.b2  # Output layer
        return self.z2
    
    def backward(self, X, y, learning_rate=0.01):
        """Backward pass: compute gradients using chain rule"""
        batch_size = X.shape[0]
        
        # Output layer gradient (MSE loss: L = (y - y_hat)^2)
        dloss_dz2 = 2 * (self.z2 - y) / batch_size  # d(loss)/d(z2)
        
        # Chain rule: propagate to W2
        dloss_dW2 = self.a1.T @ dloss_dz2  # d(loss)/d(W2) = a1^T @ dloss_dz2
        dloss_db2 = np.sum(dloss_dz2, axis=0, keepdims=True)
        
        # Chain rule: propagate to hidden layer
        dloss_da1 = dloss_dz2 @ self.W2.T  # d(loss)/d(a1)
        
        # ReLU derivative: gradient zero where z1 <= 0
        dloss_dz1 = dloss_da1 * (self.z1 > 0)  # Masked gradient
        
        # Chain rule: propagate to W1
        dloss_dW1 = X.T @ dloss_dz1
        dloss_db1 = np.sum(dloss_dz1, axis=0, keepdims=True)
        
        # Update weights (gradient descent)
        self.W2 -= learning_rate * dloss_dW2
        self.b2 -= learning_rate * dloss_db2
        self.W1 -= learning_rate * dloss_dW1
        self.b1 -= learning_rate * dloss_db1
```

## Why ReLU Instead of Sigmoid?

**Sigmoid**: $\sigma(z) = \frac{1}{1 + e^{-z}}$ ranges [0, 1], smooth
**ReLU**: $\text{max}(0, z)$ ranges [0, ∞], piecewise linear

```python
import matplotlib.pyplot as plt

z = np.linspace(-5, 5, 100)

# Sigmoid
sigmoid = 1 / (1 + np.exp(-z))
sigmoid_grad = sigmoid * (1 - sigmoid)

# ReLU
relu = np.maximum(0, z)
relu_grad = (z > 0).astype(float)

fig, (ax1, ax2) = plt.subplots(1, 2)
ax1.plot(z, sigmoid, label='σ(z)', linewidth=2)
ax1.plot(z, sigmoid_grad, label="σ'(z)", linewidth=2)
ax1.set_title('Sigmoid: Vanishing gradient problem')
ax1.legend()

ax2.plot(z, relu, label='ReLU(z)', linewidth=2)
ax2.plot(z, relu_grad, label="ReLU'(z)", linewidth=2)
ax2.set_title('ReLU: No saturation, sparse gradients')
ax2.legend()

plt.show()
```

**Why**:
- Sigmoid: Gradient → 0 at extremes (hard to train deep networks)
- ReLU: Constant gradient for z > 0 (trains faster), sparse (efficient)

## Practical Exercise

Train the SimpleNeuralNet on XOR (a problem linear classifiers can't solve):

```python
# XOR dataset
X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
y = np.array([[0], [1], [1], [0]])

net = SimpleNeuralNet(input_size=2, hidden_size=4, output_size=1)

# Train for 1000 steps
for step in range(1000):
    pred = net.forward(X)
    loss = np.mean((pred - y) ** 2)
    net.backward(X, y, learning_rate=0.1)
    
    if step % 200 == 0:
        print(f"Step {step}: loss = {loss:.4f}")

# Check: predictions should be close to [0, 1, 1, 0]
print("Final predictions:", net.forward(X).round(2).flatten())
```
