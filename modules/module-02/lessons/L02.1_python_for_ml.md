# Python for Machine Learning

## Why This Matters

Before building deep learning models, you need to be proficient with Python tools that power ML: NumPy for numerical computing, Pandas for data manipulation, and Matplotlib for visualization. These are the "languages" you'll use to transform raw data into training examples and understand model behavior.

## Key Concepts

### NumPy Arrays vs Lists

```python
# Lists are slow for numerical operations
import time
import numpy as np

# Slow: pure Python
matrix_py = [[i + j for j in range(1000)] for i in range(1000)]
start = time.time()
sum_py = sum(sum(row) for row in matrix_py)
print(f"Python: {time.time() - start:.4f}s")

# Fast: NumPy (SIMD optimization + C implementation)
matrix_np = np.arange(1000000).reshape(1000, 1000)
start = time.time()
sum_np = np.sum(matrix_np)
print(f"NumPy: {time.time() - start:.4f}s")  # ~100x faster
```

**Why**: NumPy uses C implementations and CPU-level optimizations (SIMD) that pure Python cannot match.

### Broadcasting: The Most Powerful Feature

```python
# Broadcasting automatically expands dimensions for element-wise operations
X = np.random.randn(100, 784)  # 100 images, 784 pixels each
bias = np.random.randn(784)     # 1D bias vector

# Automatically broadcasts to (100, 784)
Z = X + bias  # NO explicit loop needed!
```

**Why**: Broadcasting lets you write vectorized code (no loops) that's both readable and fast.

### Pandas for Data Loading and Cleaning

```python
import pandas as pd

# Read CSV (common data format)
df = pd.read_csv("bible_metadata.csv")

# Select columns
texts = df['text']
languages = df['language']

# Filter rows (where language == 'english')
english_df = df[df['language'] == 'english']

# Group and aggregate (prepare dataset statistics)
per_lang = df.groupby('language')['text'].apply(lambda x: len(x.sum()))
```

**Why**: ML pipelines start with messy data. Pandas makes cleaning/filtering/aggregating painless.

## Practical Exercise: Normalize Text Data

```python
import numpy as np

def normalize_text_data(texts, min_chars=10, max_chars=1000):
    """Filter and prepare texts for tokenization.
    
    Args:
        texts: List of text strings
        min_chars: Minimum characters to keep
        max_chars: Maximum characters (truncate if longer)
        
    Returns:
        List of normalized texts
    """
    normalized = []
    for text in texts:
        # Skip too short/long
        if len(text) < min_chars or len(text) > max_chars:
            continue
        # Normalize whitespace
        text = " ".join(text.split())
        normalized.append(text)
    
    # Convert to NumPy array for efficient batch processing
    return np.array(normalized, dtype=object)
```
