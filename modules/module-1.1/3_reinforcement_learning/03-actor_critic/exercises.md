# Actor-Critic - Exercises

## Exercise 1: Basic Actor-Critic
Implement simple actor-critic for CartPole. Compare with REINFORCE.

## Exercise 2: Advantage Computation
Implement different advantage estimators (TD, GAE). Show variance-bias trade-off.

## Exercise 3: Asynchronous A3C
Implement A3C (Asynchronous Advantage Actor-Critic) with multiple workers.

## Exercise 4: Continuous Control AC
Apply actor-critic to continuous control (Pendulum, Lunar Lander).

## Exercise 5: Generalized Advantage Estimation
Implement GAE for better advantage estimates. Show impact on sample efficiency.

## Exercise 6: Trust Region Policy Optimization
Implement TRPO variant of actor-critic for stable learning.

## Exercise 7: Proximal Policy Optimization
Implement PPO actor-critic. Compare with A3C on benchmark tasks.

## Exercise 8: Multi-Step Returns
Compare 1-step vs multi-step returns in actor-critic framework.

## Exercise 9: Entropy Regularization
Show impact of entropy bonus on exploration in actor-critic.

## Exercise 10: Complex Environment AC
Apply state-of-the-art actor-critic (PPO, A2C) to complex environments.
