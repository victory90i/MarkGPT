# Q-Learning - Exercises

## Exercise 1: Gridworld Q-Learning
Implement Q-Learning for a simple gridworld. Visualize learned policy and Q-values.

## Exercise 2: Exploration vs Exploitation
Implement epsilon-greedy exploration. Show learning curves with different epsilon values.

## Exercise 3: FrozenLake Environment
Use OpenAI Gym's FrozenLake. Train Q-Learning agent and achieve goal.

## Exercise 4: Convergence Analysis
Plot Q-value convergence over episodes. Show when agent stabilizes.

## Exercise 5: CartPole with Q-Learning
Discretize CartPole state space and apply Q-Learning. Compare with continuous approaches.

## Exercise 6: Function Approximation
Implement Q-Learning with neural network approximation for continuous state spaces.

## Exercise 7: Experience Replay
Implement Q-Learning with experience replay buffer. Show improvement over vanilla Q-Learning.

## Exercise 8: Double Q-Learning
Implement Double Q-Learning to reduce overestimation bias. Compare with vanilla Q-Learning.

## Exercise 9: Hyperparameter Sensitivity
Study impact of learning rate, discount factor, and epsilon decay on performance.

## Exercise 10: Multi-Environment Transfer
Train on one environment and apply to similar environment. Analyze transfer learning.
