# Deep Q-Networks - Exercises

## Exercise 1: Simple DQN
Implement DQN for CartPole. Compare with vanilla Q-Learning.

## Exercise 2: Experience Replay Buffer
Implement prioritized experience replay. Show sample efficiency improvement.

## Exercise 3: Target Network
Implement DQN with separate target network. Show stability improvement.

## Exercise 4: Atari Game Learning
Apply DQN to Atari game (Pong, Breakout) from pixels.

## Exercise 5: Double DQN
Implement Double DQN to reduce overestimation. Compare with vanilla DQN on Atari.

## Exercise 6: Dueling DQN
Implement Dueling architecture (separate value and advantage streams).

## Exercise 7: Rainbow Algorithm
Implement Rainbow combining multiple DQN improvements.

## Exercise 8: Hyperparameter Tuning
Study sensitivity to network architecture, replay buffer size, target update frequency.

## Exercise 9: Generalization
Test trained DQN on different Atari games. Analyze generalization and transfer.

## Exercise 10: Benchmark Comparison
Compare DQN, Double DQN, Dueling DQN, Rainbow on standardized benchmarks.
