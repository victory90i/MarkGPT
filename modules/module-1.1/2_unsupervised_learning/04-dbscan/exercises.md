# DBSCAN - Exercises

## Exercise 1: Basic DBSCAN
Apply DBSCAN to 2D data. Visualize clusters and identified outliers.

## Exercise 2: Parameter Sensitivity
Test different eps and min_samples parameters. Show impact on clustering results.

## Exercise 3: K-Distance Graph
Compute k-distance graphs to determine optimal eps parameter.

## Exercise 4: Comparison with K-Means
Apply both K-Means and DBSCAN to dataset with non-spherical clusters. Compare results.

## Exercise 5: Arbitrary Shapes
Apply DBSCAN to moon/spiral datasets. Show advantage over K-Means for non-convex shapes.

## Exercise 6: Outlier Detection
Use DBSCAN for outlier detection. Compare with LOF (Local Outlier Factor).

## Exercise 7: High-Dimensional Data
Apply DBSCAN to high-dimensional data. Discuss curse of dimensionality in distance computation.

## Exercise 8: Distance Metrics
Use different distance metrics with DBSCAN (Euclidean, Manhattan, custom). Compare results.

## Exercise 9: Computational Efficiency
Analyze scalability of DBSCAN with data size. Compare with K-Means performance.

## Exercise 10: Real-World Spatial Clustering
Apply DBSCAN to geographic or spatial data (GPS points, network intrusion, etc.).
