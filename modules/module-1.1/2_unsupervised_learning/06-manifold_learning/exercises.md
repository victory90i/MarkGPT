# Manifold Learning - Exercises

## Exercise 1: t-SNE Visualization
Apply t-SNE to high-dimensional data (MNIST, Iris). Visualize and explore clusters.

## Exercise 2: Perplexity Impact
Test different perplexity values in t-SNE. Show impact on visualization quality.

## Exercise 3: UMAP vs t-SNE
Compare UMAP and t-SNE on same dataset. Discuss speed and interpretation differences.

## Exercise 4: PCA + UMAP Pipeline
Apply PCA followed by UMAP. Compare with direct UMAP and t-SNE.

## Exercise 5: Local vs Global Structure
Visualize how t-SNE and UMAP preserve local vs global structure differently.

## Exercise 6: Embedding Stability
Discuss non-determinism in t-SNE. Run multiple times and analyze variation.

## Exercise 7: Computational Efficiency
Measure training time and memory for UMAP vs t-SNE on large datasets.

## Exercise 8: Cluster Validation
Use k-NN on manifold embeddings. Evaluate cluster quality from visualization.

## Exercise 9: Feature Visualization in Deep Networks
Use manifold learning to visualize learned representations from neural networks.

## Exercise 10: Real-World Exploration
Apply t-SNE/UMAP to real data for exploratory analysis and outlier detection.
