# GMM - Exercises

## Exercise 1: Basic GMM Clustering
Apply GMM to 2D data. Visualize cluster assignments and ellipses.

## Exercise 2: Soft vs Hard Assignment
Compare soft probabilistic assignments from GMM with hard K-Means assignments.

## Exercise 3: Covariance Types
Train GMM models with different covariance types (spherical, tied, diagonal, full). Compare results and complexity.

## Exercise 4: Model Selection with BIC/AIC
Use BIC and AIC criteria to determine optimal number of components.

## Exercise 5: Uncertainty Estimation
Use GMM probabilities to estimate prediction uncertainty. Compare with other uncertainty estimates.

## Exercise 6: EM Algorithm Visualization
Visualize EM algorithm convergence. Plot likelihood over iterations.

## Exercise 7: Comparison with K-Means
Apply both GMM and K-Means to same dataset. Show differences in interpretations.

## Exercise 8: Anomaly Detection with GMM
Use probability from GMM as anomaly score. Identify low-probability outliers.

## Exercise 9: Learning from Partial Labels
Use GMM with semi-supervised learning (some labeled data). Compare with fully unsupervised.

## Exercise 10: Real-World Probabilistic Clustering
Apply GMM to real data where uncertainty is important (medical diagnosis, finance).
