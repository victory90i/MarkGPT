# PCA - Exercises

## Exercise 1: Basic PCA
Apply PCA to high-dimensional data. Plot explained variance and visualize first two components.

## Exercise 2: Scree Plot Analysis
Create scree plot and cumulative variance plot. Determine optimal number of components.

## Exercise 3: Visualization with PCA
Use PCA for 2D/3D visualization of high-dimensional datasets. Compare with original space visualization.

## Exercise 4: Feature Reconstruction
Reconstruct original features from reduced components. Measure reconstruction error vs number of components.

## Exercise 5: Noise Reduction
Apply PCA for denoising noisy data. Compare signal quality before and after.

## Exercise 6: PCA Preprocessing
Use PCA as preprocessing for classification. Compare model performance with/without PCA.

## Exercise 7: Whitening and Scaling
Compare centered, scaled, and whitened PCA. Analyze differences in principal components.

## Exercise 8: Incremental PCA
Apply Incremental PCA to large datasets that don't fit in memory. Compare with standard PCA.

## Exercise 9: Face Recognition with PCA
Use PCA (eigenfaces) for face recognition. Visualize principal components as images.

## Exercise 10: Real-World Dimensionality Reduction
Apply PCA to real high-dimensional data. Compare with other methods (t-SNE, Autoencoders).
