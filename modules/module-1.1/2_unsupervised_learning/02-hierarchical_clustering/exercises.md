# Hierarchical Clustering - Exercises

## Exercise 1: Agglomerative Clustering
Apply hierarchical clustering with different linkage methods (single, complete, average, Ward).

## Exercise 2: Dendrogram Visualization
Create dendrograms and visually analyze cluster structure. Determine optimal cut threshold.

## Exercise 3: Linkage Method Comparison
Compare dendrograms from different linkage methods. Analyze differences in cluster assignments.

## Exercise 4: Distance Metrics Impact
Use different distance metrics (Euclidean, Manhattan) with hierarchical clustering. Compare results.

## Exercise 5: Optimal Cluster Number
Use dendrogram to determine optimal number of clusters. Compare with elbow method from K-Means.

## Exercise 6: Cophenetic Correlation
Calculate cophenetic correlation coefficient to assess dendrogram quality.

## Exercise 7: Large Dataset Scalability
Apply hierarchical clustering to increasingly large datasets. Measure time and memory usage.

## Exercise 8: Divisive Clustering
Implement divisive (top-down) hierarchical clustering. Compare with agglomerative approach.

## Exercise 9: Feature Preprocessing
Apply different preprocessing strategies (scaling, normalization). Show impact on hierarchical clustering.

## Exercise 10: Real-World Taxonomy Project
Apply hierarchical clustering to create taxonomies (biological, document, customer hierarchies).
