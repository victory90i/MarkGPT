# K-Means - Exercises

## Exercise 1: Basic K-Means Clustering
Apply K-Means with K=3 to a simple dataset. Visualize clusters and centroids.

## Exercise 2: Elbow Method
Compute inertia for K=1 to 10. Plot elbow curve and determine optimal K.

## Exercise 3: K-Means++ Initialization
Compare random initialization with K-means++ initialization. Measure convergence quality and speed.

## Exercise 4: Feature Scaling Importance
Train K-Means on raw and scaled features. Show impact of scaling on clustering.

## Exercise 5: K-Means for Image Compression
Apply K-Means to image pixels. Compress image using cluster centers and visualize.

## Exercise 6: Silhouette Score Analysis
Calculate silhouette scores for different K values. Identify optimal K using silhouette analysis.

## Exercise 7: Multiple Random Initializations
Run K-Means multiple times with random initialization. Analyze solution variability.

## Exercise 8: K-Means for Anomaly Detection
Use K-Means distance to cluster center as anomaly score. Identify outliers.

## Exercise 9: Mini-Batch K-Means for Large Datasets
Compare standard K-Means with Mini-Batch K-Means on large data. Measure speed and quality trade-off.

## Exercise 10: Real-World Segmentation Project
Apply K-Means to customer or document data. Evaluate clusters and extract business insights.
