# Module 10 — Training & Deploying MarkGPT
## Days 55–60 | Capstone

---

## Module Overview

This is the capstone. You have learned theory, built components, trained models on English. Now you bring it all together: train the final MarkGPT on both Biblical English and Banso text, evaluate it rigorously, and share it with the world.

### Lesson Structure (Days 55–60):
- **L55.1-L55.2** (Days 55–56): Final architecture decisions and production checklist
- **L56.1-L56.2** (Days 56–57): Quantization and ONNX model export
- **L57.1-L57.2** (Days 57–58): HuggingFace Hub deployment and cloud platforms
- **L58.1-L58.2** (Days 58–59): API serving with FastAPI and production monitoring
- **L59.1-L60.2** (Days 59–60): Community sharing and graduation capstone project

By the end of Module 10, you will:
- Train a full MarkGPT model (your final capstone)
- Fine-tune it on the Banso corpus
- Evaluate and write a model card
- Deploy it through a web API
- Reflect on what you've built and learned
