## What Is Prompting?

Prompting is the process of crafting inputs to get desired outputs from a language model. It is a powerful way to use pretrained models without retraining.

This lesson explores techniques for effective prompting and inference.

---

## Parameter Suggestions

A good starting point is:
- temperature: 0.7
- top_k: 50
- top_p: 0.9

Adjust these parameters based on the desired creativity and consistency of outputs.
