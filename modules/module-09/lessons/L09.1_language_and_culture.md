# Building Datasets for African Languages

## Why This Matters

LLMs are trained on what humans write. The internet is dominated by:
- English (55%)
- European languages (35%)
- Everything else (10%)

African languages, despite 1+ billion speakers, are nearly invisible in digital form. MarkGPT's Banso component teaches:
1. How to build datasets for under-resourced languages
2. Why linguistic diversity matters
3. The ethical dimension of AI

## The Bible as a Universal Corpus

### Why Bible Text?

1. **Parallel texts**: Bible has been translated into 2,000+ languages (more than any other book)
2. **Freely available**: Public domain or permissive licenses
3. **Culturally significant**: Used across communities
4. **Linguistic balance**: Mix of narrative, poetry, law, history

### Building a Bible Dataset

```python
from pathlib import Path
import json

def build_bible_dataset():
    """
    1. Download KJV Bible
    2. Extract verses
    3. Add metadata (book, chapter, verse)
    4. Tokenize and save
    """
    
    # KJV format: "Matthew 1:1 In the beginning..."
    # Extract into structured format
    
    dataset = {
        "book": [],
        "chapter": [],
        "verse": [],
        "text": [],
        "language": [],
    }
    
    with open("bible_kjv.txt") as f:
        for line in f:
            if line.startswith("^"):
                continue
            
            # Parse: "GEN 1 1 In the beginning"
            parts = line.rstrip().split(None, 3)
            if len(parts) == 4:
                book, chapter, verse, text = parts
                dataset["book"].append(book)
                dataset["chapter"].append(int(chapter))
                dataset["verse"].append(int(verse))
                dataset["text"].append(text)
                dataset["language"].append("english")
    
    # Save as JSON Lines (one verse per line, easy to stream)
    with open("bible_dataset.jsonl", "w") as f:
        for i in range(len(dataset["text"])):
            record = {k: dataset[k][i] for k in dataset}
            f.write(json.dumps(record) + "\n")
```

## Indigenous Languages: The Banso (Nso') Case

### Linguistic Background

**Nso'** (also spelled Lamnso', Lamndu) is spoken by ~200,000 people in:
- Menchum Division, Northwest Region, Cameroon
- Diaspora communities worldwide

### Linguistic Features

```
Word order: SOV (Subject-Object-Verb)
Tones: 3-4 lexical tones (meaning changes with pitch)
Noun classes: 15+ classes (like gender in Romance languages)

Example: 
  English: "The dog eats the bone"
  Nso':    "Fɯ́ gǔ lɯ́m ñvæ wæ"
  
  Tones: fú (dog, high tone)
         gù (it, low tone, object)
         lúm (eat)
         ñvá (bone)
         wà (comes, completes action)
```

### Building a Banso Dataset

```python
def process_banso_corpus():
    """
    Nso' data sources:
    1. Bible translation (New Testament available)
    2. Academic texts (linguistics department archives)
    3. Community-contributed content
    """
    
    # Challenge 1: Character encoding
    # Nso' uses diacritics (à, á, ǎ, ā) and special chars
    
    texts = []
    
    # 1. Bible NT (UTF-8 encoded properly)
    with open("bible_nso_nt.txt", encoding="utf-8") as f:
        texts.extend([line.strip() for line in f if line.strip()])
    
    # 2. Grammar book extracts (manually typed)
    texts.append("Ñvɯ́ mə́ bə́ nə́ fə́nə.")  # Tones are crucial!
    
    # 3. Proverbs (collected orally, transcribed)
    texts.extend([
        "Lá á bí,   á sáy.",  # When it breaks, it breaks
        "Sèn sèn tə ghe ghəi ɀɛn.",  # Even slowly, one reaches the goal
    ])
    
    # Challenge 2: Normalization
    # Different transcription conventions (tone marking, diacritics)
    
    # Challenge 3: Limited resources
    # 10,000 verses ≠ 780,000 (KJV)
    # Solution: Data augmentation and multilingual training
    
    return texts
```

### Language Preservation vs. Replacement

**Paradox**: Teaching LLMs Banso might:
- ✅ Create digital resources (corpus, model)
- ❌ Reduce incentive to learn language (everyone uses LLM)

**Reality**: Language preservation requires:
1. Community recognition of value (in Cameroon, English > Nso')
2. Educational policy (teach it in schools)
3. Economic incentives (jobs for speakers)

LLMs are a *tool*, not a solution.
